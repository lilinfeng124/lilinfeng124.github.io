java Study

构建java编译环境

1.下载

下载完安装，我安装的是百度直接搜索到的第一个程序。1.8.0_131

2.安装

基本都是下一步下一步
我装在C:\Program Files (x86)\Java\jdk1.8.0_131\lib\tools.jar;C:\Program Files (x86)\Java\环境下。其实之前也有几个版本的已经存在了，但好像并不影响。

3.设置环境变量

这一步很重要。
新建环境变量：CLASSPATH
.;C:\Program Files (x86)\Java\jdk1.8.0_131\lib\tools.jar;C:\Program Files (x86)\Java\jdk1.8.0_131\lib\dt.jar
一定要注意前面有.;不然编译完也运行不了。
Path环境变量末尾添加：
C:\Program Files (x86)\Java\jre1.8.0_131\bin
新建环境变量：java_home
C:\Program Files (x86)\Java\jre1.5.0;C:\Program Files (x86)\Java\jre1.8.0_131

4.helloworld

老规矩，一定要来个helloworld，新建一个文件Test.java，编写。

5.编译运行

进入目录：javac Test.java
会生成一个Test.class
java Test
就看到了打印结果。
